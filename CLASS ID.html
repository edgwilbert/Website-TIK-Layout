Class dan ID dalam HTML

Berikut adalah penjelasan singkat mengenai penggunaan Class dan ID dalam HTML:

  1. Class:
    • Class adalah atribut yang digunakan untuk menandai elemen-elemen yang memiliki gaya atau fungsi yang sama.
    • Dapat digunakan untuk beberapa elemen sekaligus pada satu halaman HTML.
    • Ditandai dengan tanda titik (.) di CSS saat menentukan gaya untuk class.
    • Contoh penggunaan Class:

      • <div class="container">
        <p class="text-bold">Ini adalah teks dengan class.</p>
      </div>

    • Contoh CSS untuk Class:

      • .container { padding: 20px; background-color: #f0f0f0; }
      .text-bold { font-weight: bold; color: blue; }

  2. ID:
    • ID adalah atribut yang digunakan untuk menandai elemen yang unik dan spesifik pada halaman.
    • Setiap ID hanya boleh digunakan sekali pada satu halaman HTML.
    • Ditandai dengan tanda pagar (#) di CSS saat menentukan gaya untuk ID.
    • Contoh penggunaan ID:

      • <h1 id="main-title">Judul Utama</h1>
      <p id="intro">Ini adalah paragraf pembuka.</p>

    • Contoh CSS untuk ID:

      • #main-title { font-size: 24px; color: red; }
      #intro { font-size: 18px; color: gray; }
