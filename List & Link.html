List dan Link dalam HTML

Berikut adalah rangkuman mengenai List dan Link pada HTML:

  1. List:
    • List adalah elemen HTML yang digunakan untuk membuat daftar atau poin-poin.
    • Ada dua jenis list utama:
      • Ordered List (<ol>): digunakan untuk membuat daftar bernomor.
      • Unordered List (<ul>): digunakan untuk membuat daftar berpoin (bullets).
    • Contoh Ordered List:

      • <ol>
        <li>Item 1</li>
        <li>Item 2</li>
      </ol>

    • Contoh Unordered List:

      • <ul>
        <li>Item A</li>
        <li>Item B</li>
      </ul>

  2. Link:
    • Link adalah elemen HTML yang digunakan untuk membuat tautan atau hyperlink ke halaman lain atau file tertentu.
    • Link dibuat menggunakan tag <a> dengan atribut href yang menunjuk ke URL tujuan.
    • Contoh Link:

      • <a href="https://www.example.com">Kunjungi Example.com</a>
