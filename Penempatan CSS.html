Penempatan CSS pada HTML

Berikut adalah penjelasan mengenai Penempatan CSS yang dapat digunakan pada HTML:

  1. Inline CSS:
    • Inline CSS diterapkan langsung pada elemen HTML menggunakan atribut style pada tag elemen.
    • Cocok untuk menerapkan gaya khusus pada elemen tertentu tanpa mempengaruhi elemen lain.
    • Contoh kode Inline CSS:

      • <p style="color: red; font-size: 20px;">Ini adalah teks berwarna merah.</p>

  2. Internal CSS:
    • Internal CSS ditulis di dalam tag <style> yang biasanya ditempatkan di bagian <head> pada dokumen HTML.
    • Cocok untuk mengatur gaya seluruh halaman yang hanya berlaku pada halaman tersebut saja.
    • Contoh kode Internal CSS:

      • <head>
        <style>
          p { color: blue; font-size: 18px; }
        </style>
      </head>

  3. External CSS:
    • External CSS disimpan dalam file terpisah dengan ekstensi .css dan dihubungkan ke HTML melalui tag <link> di bagian <head>.
    • Cocok untuk mengatur gaya yang konsisten di banyak halaman, sehingga struktur kode lebih rapi dan mudah dikelola.
    • Contoh pada HTML:

      • <head>
        <link rel="stylesheet" href="style.css">
      </head>

    • Contoh pada file style.css:

      • p { color: green; font-size: 16px; }
